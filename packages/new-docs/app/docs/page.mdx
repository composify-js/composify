# Introduction

Composify is an open source toolkit for building server-driven user interfaces with ease.
It allows developers to create reusable components that can be visually composed into pages by non-developers, such as marketers or product managers.
This approach accelerates the development process, reduces bottlenecks, and enhances team productivity.
