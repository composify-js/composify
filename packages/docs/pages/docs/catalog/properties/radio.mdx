export const BasicUsage = () => {
  const [value, setValue] = useState('left');

  return (
    <PropertyControlRadio
      name="example"

        type: 'radio',
        options: [
          { label: 'Left', value: 'left' },
          { label: 'Center', value: 'center' },

# Radio Property

A **radio** property lets users pick one option from a set, displayed as a group of radio buttons. 

:::code-group
<div data-title="Preview">
  <BasicUsage />
</div>

```tsx showLineNumbers [Code]
Catalog.register('Sample', {
  component: Sample,
  props: {
    alignment: {
      label: 'Alignment',
      type: 'radio',// [!code hl]
      options: [// [!code hl]
        { label: 'Left', value: 'left' },// [!code hl]
        { label: 'Center', value: 'center' },// [!code hl]
        { label: 'Right', value: 'right' },// [!code hl]
      ],// [!code hl]
    },
  },
});
```
:::

## Required Parameters

### `options`

This is an array that defines the choices for the radio group.
Each option in the array should be an object with: 

- `label`: The text displayed next to the radio button.
- `value`: The value passed to your component when the option is selected.

```tsx showLineNumbers
Catalog.register('Sample', {
  component: Sample,
  props: {
    alignment: {
      label: 'Alignment',
      type: 'radio',
      options: [// [!code hl]
        { label: 'Left', value: 'left' },// [!code hl]
        { label: 'Center', value: 'center' },// [!code hl]
        { label: 'Right', value: 'right' },// [!code hl]
      ],// [!code hl]
    },
  },
});
```

## Optional Parameters

### `render`

f you want to create a totally custom look, you can provide your own `render` function to display the radio group however you like. 

:::code-group
<div data-title="Preview">
  <WithCustomRender />
</div>

```tsx showLineNumbers [Code]
Catalog.register('Sample', {
  component: Sample,
  props: {
    alignment: {
      label: 'Alignment',
      type: 'radio',
      options: ['left', 'center', 'right'],// [!code hl]
      render: (value, onChange) => (// [!code hl]
        <HStack gap={8}>
          <IconButton
            name="left"
            selected={value === 'left'}
            onChange={() => onChange('left')}
          />
          <IconButton
            name="center"
            selected={value === 'center'}
            onChange={() => onChange('center')}
          />
          <IconButton
            name="right"
            selected={value === 'right'}
            onChange={() => onChange('right')}
          />
        </HStack>
      ),// [!code hl]
    },
  },
});
```
:::

**Heads up**: When using a custom `render` function, the `options` array should only contain the raw values (e.g., `['left', 'center', 'right']`)
 Your `render` function is now fully in charge of displaying the labels.
